# アプリケーション概要
5チャンネルのような投稿掲示板サイト

# 機能一覧
●ユーザー登録、ログイン機能

●閲覧機能

●スレッド作成機能(ログインユーザーのみ)

●スレッド作成時のタグ付け機能

●スレッド作成時のタグのインクリメンタルサーチ機能

●スレッド検索機能

●レスポンス機能(ログインユーザーのみ)
